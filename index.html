Semantic and Non-semantic Elements

Semantic elements, often used in HTML5 are coding elements that clearly define meaning to both the browser and the developer.

Semantic HTML5 tags are used to enhance acessibility, serchability, internationalization and interoperability. These are used in place of div tags.

Examples of semantic elements include:

  • Tables
  • Forms
  • Articles

Semantic elements are not to be confused with non-semantic elements, which don't define content. The two, most common examples of non-semantic elements include:

  1. div tags
  2. span tags
